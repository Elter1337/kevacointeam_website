Keva-Stratum

Download

keva-stratum is a high-performance CryptoNote mining stratum server with a web interface, specifically tailored for Kevacoin.

Forked from monero-stratum, keva-stratum simplifies the build process and includes necessary Monero source code components, eliminating the need for an external Monero source tree. It also supports building on Windows through MSYS2.

Stratum Feature List

- Be Your Own Pool: Set up and manage your own mining pool.
- Rigs Availability Monitoring: Monitor the availability and performance of mining rigs.
- Track Stats: Keep track of accepted shares, rejected shares, and block statistics.
- Detect Sick Rigs: Easily identify underperforming or malfunctioning rigs.
- Daemon Failover List: Configure a list of failover daemons to ensure continuous operation.
- Concurrent Shares Processing: Handle multiple share submissions simultaneously for optimal performance.
- Beautiful Web Interface: Manage and monitor your mining pool with an intuitive web interface.

Features in Detail

1. Be Your Own Pool:

Allows you to set up a private mining pool, giving you control over the mining process and pool fees.

2. Rigs Availability Monitoring:

Continuously monitors the status of mining rigs, ensuring they are operational and performing optimally.

3. Keep Track of Accepts, Rejects, Blocks Stats:

Provides detailed statistics on accepted and rejected shares, as well as blocks found, helping you to monitor the pool's performance.

4. Easy Detection of Sick Rigs:

Quickly identifies rigs that are underperforming or malfunctioning, allowing for prompt maintenance or replacement.

5. Daemon Failover List:

Configures multiple daemons to ensure failover in case the primary daemon goes down, maintaining continuous mining operations.

6. Concurrent Shares Processing:

Optimizes performance by handling multiple share submissions simultaneously, reducing latency and increasing efficiency.

7. Beautiful Web Interface:

An intuitive web interface for managing and monitoring the pool, providing real-time statistics and easy configuration.

By using keva-stratum, you can set up a high-performance mining pool for Kevacoin, leveraging its robust features and user-friendly interface to manage and monitor your mining operations efficiently.
