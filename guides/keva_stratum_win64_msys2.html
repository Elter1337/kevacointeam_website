Build Keva Stratum for Windows using MSYS2

Step 1:

Download MSYS2 installer from https://www.msys2.org/: Download 

https://github.com/msys2/msys2-installer/releases/download/2024-05-07/msys2-x86_64-20240507.exe

Step 2:

Install MSYS2 from downloaded installer, skip option to run at the end so correct shell can be launched.

Step 3:

Launch mingw64 either from start menu or from command line with 'mingw64'

Step 4:

Update system, relaunching mingw64 may be required 

pacman -Syu

Step 5:

Install build requirements 

pacman -S mingw-w64-x86_64-toolchain make mingw-w64-x86_64-cmake mingw-w64-x86_64-boost mingw-w64-x86_64-openssl mingw-w64-x86_64-zeromq mingw-w64-x86_64-libsodium mingw-w64-x86_64-hidapi git unzip

Step 6:

For build support we need to use an older version of go. Here we use latest known to build the project 1.19.13 

wget https://go.dev/dl/go1.19.13.windows-amd64.zip

Step 7:

Create a directory to install Go 

mkdir /c/Go-1.19.13

Step 8:

Unzip go into the Go install directory 

unzip go1.19.13.windows-amd64.zip -d /c/Go-1.19.13

Step 9:

Export the GOROOT environtment vairiable to point to the Go install directory 

export GOROOT=/c/Go-1.19.13/go

Step 10:

Update the PATH environment variable 

export PATH=$GOROOT/bin:$PATH

Step 11:

Check that our go version matches1.19.13 

go version

Step 12:

Set GO111MODULE to 'auto' for build compatibility 

export GO111MODULE=auto

Step 13:

Clone keva-stratum repository 

git clone https://github.com/kevaone/keva-stratum.git

Step 14:

Navigate into keva-stratum directory 

cd keva-stratum

Step 15:

Make a directory to build from, here we use build 

mkdir build

Step 16:

Navigate into the build directory 

cd build

Step 17:

Run cmake with the MSYS tool chain 

cmake -G "MSYS Makefiles" ..

Step 18:

Build keva-stratum 

make

Step 19:

Update keva-stratum config 

nano config.json

Step 20:

Run keva-stratum 

keva-stratum.exe

Build file located within c:/msys2/home/'user'/keva-stratum/build. If moving to another location you need kevastratum.exe, libcnutil.dll, config.json and www directory.
