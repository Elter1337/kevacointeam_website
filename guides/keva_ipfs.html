Build Keva_IPFS for Linux

Step 1:

Make sure system is up to date 

sudo apt update

sudo apt upgrade

Step 2:

Install Git and Golang 

sudo apt install git golang

Step 3:

Clone keva_ipfs repository 

git clone https://github.com/kevacoin-team/keva_ipfs.git

Step 4:

Navigate into keva_ipfs 

cd keva_ipfs/

Step 5:

Make build directory and navigate to it 

mkdir build

cd build/

Step 6:

Build keva_ipfs 

go build ..

Step 7:

Copy config.json to build directory 

cp ../config.json .

Step 8:

Update config.json 

nano config.json

Step 9:

Launch keva_ipfs 

./keva_ipfs
