Build Kevacoin Core for Linux

Step 1:

Update system 

sudo apt update

sudo apt upgrade

Step 2:

Install build requirements 

sudo apt-get install build-essential libtool autotools-dev automake pkg-config libssl-dev libevent-dev bsdmainutils python3 cmake libsodium-dev git curl

Step 3:

Clone kevacoin respository 

git clone https://github.com/kevacoin-team/kevacoin.git

Step 4:

Navigate to the depends directory within kevacoin 

cd kevacoin/depends/

Step 5:

Run make targeting Linux 

make HOST=x86_64-pc-linux-gnu

Step 6:

Navigate up a directory 

cd ..

Step 7:

Run autogen.sh 

./autogen.sh

Step 8:

Run configure, targeting the Linux depends build 

CONFIG_SITE=$(pwd)/depends/x86_64-pc-linux-gnu/share/config.site ./configure --prefix=/

Step 9:

Build Kevacoin 

make

Step 10:

Build deployment package 

make deploy

Step 11:

Install built Kevacoin 

sudo make install
