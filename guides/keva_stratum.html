Build Keva Stratum for Linux

Step 1:

Update system 

sudo apt update

sudo apt upgrade

Step 2:

Install build requirements 

sudo apt-get install build-essential libtool autotools-dev automake pkg-config libssl-dev libevent-dev bsdmainutils python3 cmake libsodium-dev libboost-system-dev libboost-filesystem-dev libboost-chrono-dev libboost-program-options-dev libboost-test-dev libboost-thread-dev

Step 3:

Install Golang and Git 

sudo apt-get install golang git

Step 4:

Clone keva-stratum 

git clone https://github.com/kevaone/keva-stratum.git

Step 5:

Enter keva-stratum directory 

cd keva-stratum/

Step 6:

Make build directory 

mkdir build

Step 7:

Navigate to build directory 

cd build/

Step 8:

Export GO111MODULE to auto for build compatibility 

export GO111MODULE=auto

Step 9:

Configure make 

cmake ..

Step 10:

Build keva-stratum 

make

Step 11:

Check build outputs present 

ls -l

Step 12:

Update config.json 

nano config.json
